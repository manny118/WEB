Table with <thead>,<tbody> and <tfoot>,

Use the <thead>,<tbody> and <tfoot> elements for large tables that may include a scroll bar or extend over several print pages.

Note the nesting order of the <thead>,<tbody> and <tfoot> elements. In the past the <tfoot> element was placed between the <thead>,<tbody> elements. See W3SCHOOLS for an example of the 'old' way example as found on the 05/07/2017. Now the correct nesting order is <thead>,<tbody> and <tfoot>. The new order will somehow make it easier to code tables dynamically.

As we noticed today (05/07/2017), the old way as stated on W3Schools and the new way will both render correctly in the browser. However, only the new way will pass the W3C validation test.
