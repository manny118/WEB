INTRODUCTION TO JAVA

JAVA first appeared in May 23, 1995. It was designed by James Gosling. JAVA is owned by Oracle.

Java is one of the most widely used computer languages all over the world. It is an object-oriented language which is based on 'write once run anywhere. It has over 9 million developers.

The current version is the JAVA 8 which was released on 18/03/2014. The syntax of JAVA was influenced by the c++ programming language.

One of the principles of JAVA is that it must be simple, object-oriented, and familiar. Also it must be robust and secure.

INTRODUCTION TO SQL

The Structured Query Language(SQL) is a computer language which is used for database communication. SQL allows you to access and manipulate databases.

SQL is widely used for data manipulation, data storage as well as querying data in relational databases. The original version was called SEQUEL(Structured English Query Language). SQL was introduced by the Oracle corporation in 1979.

INTRODUCTION TO HTML

The Hypertext Markup Language(HTML) is a standard mark-up language which is used for creating webpages. HTML is widely used for creating a structure for webpages. It is used with CSS(Cascading Style Sheets) which is used primarily for styling HTML content as well as Javascript which is a weakly typed language used for client-side programming.
