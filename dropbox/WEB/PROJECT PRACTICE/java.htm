CHAPTER 1: JAVA BASICS

I hope you ready and excited to write your first java code. Create a java class and type the following:

public class Exercise1 {
  public static void main(String[] args) {
   System.out.println("Hello World");
  }
}

Please ensure you type the syntax correctly. Do not forget to end the line with a semicolon. The console should display "Hello World".

System.out.println(); is used to print to console.

In java, classes are the building blocks we use to write our programs. A class is a template which is used to create an object. That brings us to the concept of objects. An object is a template of a class. We can create as many templates as we want from a class.

Data Types in Java

Data Type Explanation
Integer It is a 32-bit data type.
String The string datatype is one of the most used data types in Java
Boolean It can be used to store a true or false value which is a 0 or 1 bit.
Character It can be used to store a single character or symbol.
Double This is a 64-bit data type. It is the default type for decimal numbers.
Float This is a 32-bit data type which can be used for storing floating points.
