Data types in Java

  1. Integer:

    This data type is used for numerics with no fractions or decimals allowed.

  2. String:

    Strings are used for groups of characters. They are widely used in java.

Why do we need data types?

Every programming language has it's data types. They share some data types although these data types may vary from one language to anohter.

Data types are used as references for variables.
