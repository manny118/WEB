CHAPTER 1

SQL uses a DML(Data Manipulation Language), DDL(Data Definition Language) and DCM(Data Control Language). DDL commands are used for defining data structures. DML commands can be used for data manipulation

In this chapter we will focus mainly on DDL commands. They include create, alter, drop and truncate.

  1. Create: It can be used to create a database objects such as a tables, stored procedures, views, triggers, e.t.c
  2. Alter: It can be used to make changes to the structure of a databse object.
  3. Drop: It can be used to delete an object from a database.
  4. Truncate: It can be used to empty a table. All allocated spaces will be removed but the structure of the table will remain.

The following can be used to write DDL statements:

  • CREATE TABLE table_name(col1 datatype, col2 datatype);
  • ALTER TABLE table_name DROP COLUMN col1;
  • DROP TABLE table1;
  • TRUNCATE TABLE table_name;

    • CHAPTER 2: DML

    In this chapter, I will introduce you to some SQL's DML statements. DML stands for Data Manipulation Language. They are used for modify data. They can be used to make changes to existing data such as adding or deleting data.

    1. Update: It's used to modify existing rows. You can update one row at a time.
    2. Delete: The delete command is to delete one or more rows from a table.
    3. Insert: It can be used to add new rows to an existing table.
    4. Select: It can be used to retrieve rows from a table.
    5. Merge: It can be used to perform multiple DML operations.

    The following are some examples of DML statements.

    1. SELECT * FROM table1
    2. UPDATE table1 SET col1 = 4 WHERE ID = 5;
    3. INSERT INTO table1(col1, col2) VALUES(4, 6);
    4. DELETE FROM table1 WHERE ID = 1;
