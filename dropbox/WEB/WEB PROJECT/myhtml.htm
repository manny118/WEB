CHAPTER 1: HTML ELEMENTS

An HTML element has a start tag and end tag. Try the following:
<html>
 <head>
  <title>My title</title>
 </head>
 <body>
  <h2>hey</h2>
  <p>hey</p>
 </body>
</html>

This is a paragraph.

It is important to note that every HTML document must have a 'head' tag and a 'body' tag.
